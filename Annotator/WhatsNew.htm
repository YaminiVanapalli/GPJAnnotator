The Annotator

 

 

 

 

 

 

 

 

New Look…

The colors may have changed, but all the existing functionality, and more, is still in there.  Nose around and you’ll see some of the new features.

 

A change in the Comments Interface…

A new feature in the Comments Interface allows you to attach a document along with your comment, including an Excel spreadsheet, a Word document, a text file, a pdf file, a jpeg, or others.  This will allow you to attach, for example, an existing Excel spreadsheet listing the vehicles scheduled to go to a particular Auto Show, or a list of Client-provided equipment for a Trade Show, instead of re-typing the list as a comment.

 

 

Graphics Handler Changes…

The ability to download or email multiple files has been added.  Now in the thumbnail viewer interfaces there are new “Download…” and “Email Copy…” options.  By clicking the hypertext, a pop-up offers the option to “Activate Mode”.  Once clicked, checkboxes appear adjacent to all the thumbnail images.  Check the ones you wish to email or download, then re-click the hypertext and select the “Download Selections…” or the “Email Selections…” to complete.

 

From an open image file, using either the right-click menu or the “Menu” button pop-up, you now have the ability to email the displayed file to a non-Annotator user. 

 

Another new feature in the Email interface is the ability to recall past email addresses you have used.  Past recipients for the current Client exist on a drop-down list.  Click it to pick from the list, or upon typing, a type-ahead feature will attempt to auto-fill based on your past emails.

 

Navigation through multiple screens of thumbnail images has changed slightly, offering “First|Previous|Next|Last” options.  Plus, there is a text list option when there are more then twenty images in a selection. 

 

There is a new option to “Show Approved Graphics Only”.  Checking it will hide all non-Approved files.

 

 

Digital Image Library Changes…

Through the Graphics Importer interface, Digital Image Librarians can now import PowerPoint files and Video Files (.MPG, .AVI and .MOV formats) into the DIL.  The files are viewable or downloadable from the DIL, but Users should be aware performance will be affected by Network traffic and available bandwidth.  The view capabilities are intended to provide the User with the ability to browse for the file they need, then download it to their local machine for demonstrations.  Be aware, the file sizes can be fairly large and download times could be lengthy during high-traffic times.
