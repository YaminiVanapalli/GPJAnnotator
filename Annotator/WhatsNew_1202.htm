New URL…

The hyperlinks from Global Links, the GPJ Employee Portal and automated emails coming from the Space Plan Application all point to a new URL:  https://gpjapps.gpjco.com.  This is now pointing to a new Secure Gateway solution that allows us to eliminate many of the external firewall issues that have been a problem in the past.  If you prefer to type in this address in your browser, be sure to notice the “s” in “https”, this directs you to a Secure HTTP site that uses a VeriSign Secure Server Certificate. The login interface has changed, but for internal Users, you are still using your Novell login and Novell password to enter.  For Clients coming in, the interface may look a little different, but their password will be the same as it was before.  After logging in, a GPJ Application Portal screen displays a “GPJ Annotator” icon.  Click it to enter.

 

Space Plan PDFs are now available!

As new Space Plans are posted to the Annotator, a downloadable PDF is also posted.  When viewing a plan, you may select to download the PDF by clicking on the “Viewer Menu…” button and selecting “Download PDF of Current Floorplan…”.  This option is also available on the right-click menu of the Space Plan image.  PDF files are only available for plans that have been posted since November 1, 2002.

 

Another new PDF option is the ability to email a copy of the PDF to someone.  The email interface pops up after selecting “Email PDF of Current Plan…” from the “Viewer Menu” or the right-click menu.  The easy-to-use email interface gives you several options that you can take advantage of.

 

More Secure Connection…

If you are a User who tends to open a file and leave it open for a long period of time, you have probably received a disconnect error in the past that booted you out of the application.  This database “timeout” should no longer occur.  We prefer that you not leave the Annotator open when you are not using it, thereby making the files available for other Users, but if you do need to, the Annotator should not kick you out due to inactivity.
